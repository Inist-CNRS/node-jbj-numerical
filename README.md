# Numerical text processing for JBJ

Add filters to [JBJ](https://github.com/castorjs/node-jbj).

# Installation

With [npm](http://npmjs.org) do:

    $ npm install jbj-numerical

# Documentation

## Usage

```javascript
var JBJ = require('jbj');

JBJ.use(require('jbj-numerical'));
```
